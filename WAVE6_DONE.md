# Wave 6 — углубление банков, аудит вариативности и flow-smoke проверки

## Что сделано

### 1) Добавлен `wave6_boosters.js`
Новый слой контентных бустеров усиливает темы с низкой вариативностью без переписывания базовых генераторов.

Подход:
- тема не ломается и сохраняет старый генератор;
- поверх него добавляется extra-пул вопросов;
- в рантайме часть вопросов берётся из extra-банка.

### 2) Усилены 20 тем в приоритетных зонах
Добавлены extra-вопросы по темам:

- **1 класс**
  - Математика → `Фигуры`
  - Русский → `Заглавная буква`
  - Окружающий мир → `Времена года`, `Живая природа`

- **2 класс**
  - Математика → `Единицы измерения`
  - Русский → `Перенос слов`
  - Окружающий мир → `Родная страна`

- **8 класс**
  - Обществознание → `Человек и общество`, `Экономика`, `Право`

- **9 класс**
  - Обществознание → `Человек и общество`, `Экономика`, `Право`

- **10 класс**
  - Русский → `Словарные слова`
  - Информатика → `Алгоритмы`

- **11 класс**
  - История → `Россия 1990–2000-е`, `Мир во 2-й пол. XX–XXI в.`
  - Обществознание → `Политология`, `Экономика`, `Право`

### 3) Добавлен аудит вариативности по темам
Новый скрипт `topic_coverage_audit.js`:
- прогоняет каждую тему 120 раз;
- считает число уникальных формулировок вопроса;
- пишет:
  - `TOPIC_COVERAGE_AUDIT.md`
  - `TOPIC_COVERAGE.json`

Это позволяет дальше наращивать банки не на глаз, а по цифрам.

### 4) Добавлен flow-smoke уровень проверки
Новый `flow_smoke_check.js` теперь прогоняет:
- grade1–grade11:
  - открыть предмет,
  - открыть тему,
  - стартовать тренажёр,
  - ответить,
  - проверить сохранение `progress/daily/streak`.
- `diagnostic.html`:
  - старт `mathall`,
  - ответ на вопрос,
  - переход к следующему вопросу.

### 5) Усилен релиз-валидатор
`validate_release.py` теперь проверяет:
- syntax,
- SW assets,
- legacy links,
- runtime smoke,
- **flow smoke**,
- curriculum audit,
- **topic coverage audit**,
- content guards,
- guard: каждая усиленная тема должна иметь не меньше **10** уникальных формулировок.

### 6) SW обновлён под новый runtime-ассет
`sw.js`:
- cache version поднята до `trainer-v5`;
- в precache добавлен `wave6_boosters.js`.

- все усиленные темы теперь имеют **10+** уникальных формулировок;

## Что логично брать дальше

Следующая контентная волна уже хорошо видна из coverage-аудита:
- grade2 → `Безударные гласные`, `Парные согласные`, `Животные`
- grade5–7 → история / биология / география
- grade9 → `СССР 1920–30-е`, `Мир в XX в.`, `ССП и БСП`
- grade11 → `Орфография`, `Волновая оптика`, `Углы и перпендикулярность`

То есть следующий шаг — добивать уже не случайные темы, а **самые слабые по coverage-метрике**.
