# Wave 21 — progress UX

## Что сделано

### 1) Session progress shell
- Добавлен patch-layer `wave21_progress.js`.
- Во всех `grade1_v2.html ... grade11_v2.html` появился session progress block.
- На экране тренировки теперь показываются:
  - номер вопроса,
  - прогресс-бар,
  - режим/контекст сессии,
  - quick-action для избранного.
- Для обычной тренировки введена мягкая мини-сессия `20` вопросов без принудительного завершения.

### 2) Повтор ошибок текущей сессии
- Добавлен session-scoped error queue.
- После завершения обычной сессии появляется кнопка `🔁 Повторить ошибки этой сессии`.
- Повтор идёт не по слабым темам вообще, а по реальным ошибкам только что завершённого теста.
- В queue сохраняются:
  - текст вопроса,
  - варианты,
  - правильный ответ,
  - выбранный ответ,
  - hint,
  - subject/topic context.

### 3) Избранные темы
- Добавлено хранение `trainer_favs_<GRADE>`.
- Текущую тему можно добавить/убрать из избранного из theory/play UX.
- На главном экране появился блок `★ Избранные темы`.
- В списке тем избранные помечаются `★`.

### 4) Продолжить тему
- Добавлено хранение `trainer_last_topic_<GRADE>`.
- На главном экране появилась карточка `⏯ Продолжить тему`.
- Возврат идёт в последнюю тренировочную тему текущего класса.

### 5) Случайная тема
- На главном экране и в теории добавлена кнопка `🎲 Случайная тема`.
- Выбор идёт только по доступным предметам/темам текущего класса.

### 6) Snapshot autosave + resume
- Добавлен autosave незавершённой сессии в `trainer_session_snapshot_<GRADE>`.
- Snapshot сохраняется:
  - в начале сессии,
  - затем каждые 3 отвеченных вопроса,
  - с сохранением текущего вопроса и таймера.
- На главном экране появляется карточка `▶ Вернуться к сессии`.
- Resume восстанавливает:
  - subject/topic,
  - режим (обычный/диагностика/молния/error-review),
  - таймер,
  - текущий вопрос,
  - counters/stats,
  - queue повтора ошибок.

### 7) Инфраструктура и проверки
- `sw.js` обновлён до `trainer-v18`.
- `wave21_progress.js` добавлен в precache.
- Добавлен `wave21_progress_audit.js`.
- Обновлён `browser_e2e_smoke.py`:
  - проверка session progress shell,
  - проверка favourites + continue,
  - проверка session snapshot.
- Обновлён `validate_release.py`:
  - hook-check для wave 21,
  - audit-check для wave 21,
  - guard-check для feature-set wave 21.
